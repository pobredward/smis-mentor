# 🎨 모바일 프로필 UI 웹 통일 완료!

## ✅ 변경 사항

### Before (기존 모바일 UI)
- 기수 정보가 카드 그리드 형태
- 역할(role) 배지 표시
- 전화번호, 학교, 전공이 간단하게 표시

### After (웹 통일 UI)
- ✅ 웹과 동일한 레이아웃 구조
- ✅ "SMIS 캠프 참여 이력" 섹션으로 통일
- ✅ 리스트 형태로 기수 표시 (웹과 동일)
- ✅ 활성 기수 강조 (파란색 테두리)
- ✅ 배지 시스템 통일 (활성, 코드, 역할, 반)

## 🎯 배지 시스템

### **활성 배지** (파란색)
- 현재 선택된 기수 표시
- `backgroundColor: #3b82f6`
- `color: #ffffff`

### **코드 배지** (회색)
- 기수 코드 (E27, S26 등)
- `backgroundColor: #f1f5f9`
- `borderColor: #cbd5e1`

### **역할 배지** (회색)
- 담임, 수업, 서포트 등
- `backgroundColor: #f1f5f9`
- `borderColor: #cbd5e1`

### **반 배지** (하늘색)
- 반 코드 (E03 등)
- `backgroundColor: #dbeafe`
- `borderColor: #93c5fd`

## 📊 기수 선택 동작

1. **일반 기수 (비활성)**
   ```
   ┌─────────────────────────────┐
   │ 26기 여름 영어캠프          │
   │ [S26] [수업]                │
   └─────────────────────────────┘
   ```

2. **활성 기수 (클릭됨)**
   ```
   ┌─────────────────────────────┐ ← 파란색 테두리 (2px)
   │ 27기 겨울 영어캠프          │ ← 파란색 배경 (#eff6ff)
   │ [활성] [E27] [담임] [E03]  │
   └─────────────────────────────┘
   ```

3. **클릭 시**
   - activeJobExperienceId 업데이트
   - "기수가 변경되었습니다" 알림
   - 캠프 탭 자동 업데이트

## ✨ 개선된 UX

1. **명확한 계층 구조**
   - 헤더 → 프로필 → 참여 이력 → 학교 정보 → 로그아웃

2. **일관된 디자인 언어**
   - 웹과 동일한 섹션 카드
   - 웹과 동일한 배지 시스템
   - 웹과 동일한 색상 체계

3. **직관적인 인터랙션**
   - 활성 기수는 파란색으로 강조
   - 클릭 가능한 기수는 시각적으로 구분
   - 즉시 피드백 제공
