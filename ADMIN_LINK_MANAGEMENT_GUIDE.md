## 🔗 링크 추가하기

### 교육 링크 추가 (노션)

1. **교육 탭**으로 이동
2. 맨 왼쪽 **+ 버튼** 클릭
3. 정보 입력:
   - **제목**: 예) `교육일정`, `수업자료`
   - **URL**: 노션 공유 링크 붙여넣기
4. **추가** 버튼 클릭

### 시간표 링크 추가 (구글시트)

1. **시간표 탭**으로 이동
2. 맨 왼쪽 **+ 버튼** 클릭
3. 정보 입력:
   - **제목**: 예) `01/25(월)`, `정규`, `스팀`
   - **URL**: 구글시트 공유 링크 붙여넣기
     ```
     예시: https://docs.google.com/spreadsheets/d/e/2PACX-...
     ```
4. **추가** 버튼 클릭

### 인솔표 링크 추가 (구글시트)

1. **인솔표 탭**으로 이동
2. 맨 왼쪽 **+ 버튼** 클릭
3. 정보 입력:
   - **제목**: 예) `01/25 입소`, `02/13 퇴소`
   - **URL**: 구글시트 공유 링크 붙여넣기
4. **추가** 버튼 클릭

## 🗑️ 링크 삭제하기

1. 삭제할 **토글 버튼**을 **길게 누르기** (1-2초)
2. 확인 팝업에서 **삭제** 버튼 클릭

⚠️ **주의**: 삭제된 링크는 복구할 수 없습니다!

## 🔄 링크 순서 변경하기

> 🚧 **향후 기능**: 드래그 앤 드롭으로 순서 변경 가능 (현재는 삭제 후 재추가로 순서 조정)

**현재 방법:**
1. 순서를 바꾸고 싶은 링크 삭제
2. 원하는 위치에 다시 추가
   - 새로 추가된 링크는 항상 맨 뒤에 추가됨

## ❓ 문제 해결

### Q1. "등록된 링크가 없습니다" 메시지가 표시됩니다
**해결방법:**
1. 관리자 계정으로 로그인되어 있는지 확인
2. **+ 첫 링크 추가하기** 버튼 클릭하여 링크 추가

### Q2. 추가한 링크가 다른 사용자에게 보이지 않습니다
**해결방법:**
1. 앱을 완전히 종료 후 재시작
2. 해당 기수에 속한 사용자인지 확인
3. Firebase 네트워크 연결 확인

### Q3. URL을 입력했는데 "링크 추가에 실패했습니다" 오류가 발생합니다
**해결방법:**
1. URL이 올바른지 확인 (`https://`로 시작)
2. 노션 링크는 **공유 설정**이 "웹에서 공개"로 되어있는지 확인
3. 구글시트 링크는 **게시된 링크**를 사용하는지 확인
   - 파일 > 공유 > 웹에 게시

### Q4. 링크를 삭제했는데 다시 나타납니다
**해결방법:**
1. 앱 캐시 지우기
2. 앱 재시작
3. 문제가 지속되면 개발팀에 문의

## 💡 팁

### 노션 링크 가져오기
1. 노션 페이지 우측 상단 **공유** 버튼
2. **웹에서 공개** 활성화
3. **링크 복사** 클릭
4. 앱에서 붙여넣기

### 구글시트 링크 가져오기
1. 구글시트 열기
2. **파일** > **웹에 게시**
3. **링크** 선택 > **게시**
4. 생성된 링크 복사
5. 앱에서 붙여넣기

### 기수별 링크 관리
- 각 기수는 독립적인 링크 세트를 가집니다
- E27 기수의 링크는 E27에 속한 사용자만 볼 수 있습니다
- S28 기수 추가 시 해당 기수의 링크를 별도로 추가해야 합니다
