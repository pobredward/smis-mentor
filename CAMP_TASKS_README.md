# 캠프 "업무" 탭 구현 완료 🎉

## 📋 구현 내용

### 1. **데이터 구조** (`packages/shared/src/types/camp.ts`)

```typescript
- Task: 업무 메인 타입
- TaskSchedule: 일정 정보 (시작일/마감일, 시간 옵션)
- TaskDuration: 소요 시간 (분/시간 단위)
- TaskAttachment: 첨부파일 (이미지/영상/링크/파일)
- TaskCompletion: 완료 기록 (누가, 언제, 어떤 역할로)
```

### 2. **Service Layer** (`packages/web/src/lib/taskService.ts`)

#### 주요 기능:
- ✅ `createTask()`: 업무 생성
- ✅ `getTasksByCampCode()`: 캠프별 업무 목록 조회
- ✅ `updateTask()`: 업무 수정
- ✅ `deleteTask()`: 업무 삭제 (첨부파일 포함)
- ✅ `toggleTaskCompletion()`: 업무 완료/취소 토글
- ✅ `uploadTaskImage()`: 이미지 업로드 (썸네일 자동 생성)
- ✅ `uploadTaskFile()`: 파일 업로드
- ✅ `formatScheduleDisplay()`: 일정 표시 포맷팅
- ✅ `calculateDday()`: D-day 계산 (시간 고려)
- ✅ `formatDuration()`: 소요 시간 표시

### 3. **UI 컴포넌트**

#### TaskContent (`packages/web/src/components/camp/TaskContent.tsx`)
- ✅ 카테고리 탭 (캠프 전 준비, 1주차~4주차, 캠프 후 정리)
- ✅ 역할별 필터링 (자신의 groupRole에 해당하는 업무만 표시)
- ✅ 우선순위별 정렬 (높음 > 보통 > 낮음)
- ✅ 완료/미완료 분리 표시
- ✅ 업무 카드 컴포넌트
  - 체크박스로 완료 토글
  - 우선순위 아이콘 (🔴🟡⚪)
  - 일정 및 D-day 표시
  - 소요 시간 표시
  - 첨부파일 미리보기
  - Admin 완료 현황 확인

#### TaskFormModal (`packages/web/src/components/camp/TaskFormModal.tsx`)
- ✅ 업무 추가/수정 모달
- ✅ 기본 정보 입력 (제목, 설명, 대상 역할, 카테고리, 우선순위)
- ✅ **유연한 일정 설정**
  - 시작일/마감일 독립 설정
  - 시간 선택적 지정 (체크박스로 활성화)
  - 실시간 미리보기
- ✅ **소요 시간 설정**
  - 숫자 입력 (0.5 단위)
  - 단위 선택 (분/시간)
  - 표시 텍스트 자동 생성
- ✅ **첨부파일 관리**
  - 이미지/파일 업로드 (Firebase Storage)
  - 링크 추가 모달
  - 업로드 상태 표시
  - 첨부파일 제거

### 4. **페이지 연결**
- ✅ `/camp` 페이지에 "업무" 탭 추가
- ✅ TaskContent 컴포넌트 임포트 및 연결

## 🚀 사용 방법

### 1. 업무 추가 (Admin)
```
1. 캠프 탭 → 업무 탭 이동
2. "새 업무 추가하기" 버튼 클릭
3. 모달에서 정보 입력:
   - 제목, 설명
   - 대상 역할 선택 (다중 가능)
   - 카테고리 & 우선순위
   - 일정 설정 (시간 선택적)
   - 소요 시간 설정
   - 첨부파일 추가
4. "추가하기" 클릭
```

### 2. 업무 확인 및 완료 (일반 유저)
```
1. 캠프 탭 → 업무 탭 이동
2. 카테고리 선택 (캠프 전 준비, 1주차 등)
3. 자신의 역할에 해당하는 업무만 표시됨
4. 업무 확인:
   - 제목, 설명 읽기
   - 일정 및 D-day 확인
   - 소요 시간 참고
   - 첨부파일 확인
5. 체크박스 클릭하여 완료 처리
```

### 3. 완료 현황 확인 (Admin)
```
1. 각 업무 카드 하단에 완료 현황 표시
2. "완료: N명" 클릭
3. 완료자 및 미완료자 리스트 확인
```

## 📊 일정 설정 예시

### 예시 1: 간단한 업무 (일자만)
```
마감일: 2026년 3월 10일 (시간 없음)
→ 표시: "📅 마감: 3월 10일 | 🔴 D-3"
```

### 예시 2: 시간이 중요한 업무
```
시작: 2026년 3월 8일 09:00
마감: 2026년 3월 8일 13:00
→ 표시: "📅 3월 8일 09:00 ~ 3월 8일 13:00 | ⏰ 5시간 남음"
```

### 예시 3: 기간이 있는 프로젝트
```
시작: 2026년 4월 1일
마감: 2026년 4월 5일
소요 시간: 3시간
→ 표시: "📅 4월 1일 ~ 4월 5일 | ⏱️ 약 3시간 | 🔴 D-25"
```

## ✨ 특징

### 1. 유연한 일정 관리
- 시작일/마감일 독립 설정 가능
- 시간을 선택적으로 지정 (일자만 or 일자+시간)
- 실시간 미리보기로 표시 확인

### 2. 역할 기반 필터링
- 각 유저는 자신의 groupRole에 해당하는 업무만 확인
- Admin은 모든 업무 및 완료 현황 모니터링

### 3. 멀티미디어 지원
- 이미지 업로드 (썸네일 자동 생성)
- 파일 업로드 (PDF, DOCX 등)
- 링크 추가 (YouTube, Google Drive 등)

### 4. 실시간 D-day 계산
- 시간이 설정된 경우 정확한 시간 단위 계산
- 마감일에 따른 상태 표시 (D-N, 오늘 마감, N일 지남)

### 5. 완료 현황 추적
- 누가, 언제, 어떤 역할로 완료했는지 기록
- Admin이 역할별 완료율 확인 가능
